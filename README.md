#UserInterfaceExtensons#
- Email: Modifier uses UIKit + MessageUI to present a pre-filled email form. Configure email via the `ComposeMailData` and `AttachmentData` structs.
- Animation: Modifier to disable animations on MacOS.  Some animations fail to work prooperly on macOS.
- Glass: Use glass on iOS 26+ and the older UI on older devices.
- GestureSupport: Adds support for shake-to-undo to SwiftUI.
- Hardware: Use enums as keyboard shortcuts.
